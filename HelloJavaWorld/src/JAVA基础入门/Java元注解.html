注解定义：public @interface ClassName {}

一、元注解：可以注解到注解的注解
@Documented:javadoc会提取带有Documented修饰的注解类修饰的对象
@Retention(value = RetentionPolicy.*)：只能用于修饰Annotation定义
RetentionPolicy.CLASS:默认值，注解记录在class文件中，只会保留到编译阶段，不会加载到JVM，运行时不可获取
RetentionPolicy.RUNTIME:注解记录在class文件中，运行时可获取，可反射
RetentionPolicy.SOURCE:注解记录在源码中，不会被编译
@Target(value = ElementType.*),例如：@Target(value = { ElementType.ANNOTATION_TYPE,ElementType.CONSTRUCTOR })
@Inherited
@Repeatable:JDK8新特性 
@interface Roles {
	Role[] value();
}
@Repeatable(Roles.class)
// 一个容器注解,它里面必须要有一个 value 的属性，属性类型是一个被 @Repeatable 注解过的注解数组，注意它是数组
@interface Role {
	String role() default "";
}
//美人鱼即是鱼又是人
@Role(role = "fish")
@Role(role = "people")
public class Mermaid {

}
